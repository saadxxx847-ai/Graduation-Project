## 二、最可能的根因（95%概率）
**NI（Normalization Independence）归一化的统计量计算方向完全反了**

你的NI设计初衷是"每个测试窗口独立归一化"，但代码中很可能是：
```
❌ 错误的NI流程：
原始历史数据 → 减去均值 → 除以标准差 → 输入模型
模型输出预测值 → 加上均值 → 乘以标准差 → 最终预测
```
但**均值和标准差的计算方向搞反了**：
- 你计算的是**归一化后数据的均值和方差**，而不是原始数据的
- 或者更常见的：**反归一化时用了 (x * σ + μ) 而不是 (x * σ + μ) 的正确顺序**？不，等一下，更可能的是：

**真正的致命错误**：
```
✅ 正确的归一化：x_norm = (x - μ) / σ
✅ 正确的反归一化：x = x_norm * σ + μ

❌ 你代码中很可能写成了：x = x_norm * μ + σ
```
或者：
```
❌ 反归一化时用了训练集的全局均值和方差，而不是每个测试窗口独立的统计量
```
